De tekst in horizontaal gecentreerd doordat er op het div element staat text-align: center

De propertie vertical align heeft geen invloed op de tekst omdat dit enkel invloed heeft op het block zelf

Vertical align zorgt ervoor dat je de positie van je block kunt aanpassen

om 1 regel tekst verticaal te centreren moet je gebruik maken van line-height, deze plaats je op de height van je element

om meerdere regels tekst verticaal te centreren kun je gebruik maken van table-cell, hiervoor plaats je de display op display-cell en dan kan je wel vertical-align gebruiken.

De class tall krijgt voorrang omdat deze specifieker is dan de class area
