Antwoorden op de opdrachten:

Vragen demo 1 globale variabelen

  • De data blijft niet behouden als je de pagina herlaadt
  • De data blijft niet behouden als je het tabblad sluit en opnieuw opent
  • Nee, de data op de 2 tabbladen is verschillend

Vragen demo 2 session storage

  • Ja, de data blijft behouden als je het tabblad sluit en opnieuw opent
  • Als je de pagina herlaadt, blijft de data ook behouden
  • De data word niet gedeeld als je het op verschillende tabbladen opent
  • Nee, een andere pagina kan niet aan de data als je ze in hetzelfde tabblad opent

Vragen demo 3 local storage

  • Ja, de data blijft behouden als je het tabblad sluit en opnieuw opent
  • Als je de pagina herlaadt, blijft de data ook behouden
  • Ja, de data blijft ook behouden als je de browser sluit en opnieuw opent
  • Als je de pagina in een andere browser opent behoudt het deze data niet
  • Als ik een 2de tablad open, blijft die data ook behouden
  • Nee, een andere pagina kan niet aan de data als je ze in hetzelfde tabblad opent
